# @mdxg/web

Web app for the [Markdown Experience Guidelines (MDXG)](https://github.com/vercel-labs/mdxg) specification.

## Getting Started

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
